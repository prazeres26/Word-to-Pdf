# 📝 word-para-pdf

Uma ferramenta gratuita e online para converter arquivos **Word (.docx)** em **PDF** e vice-versa.

## 🚀 Funcionalidades

- ✅ Converter `.docx` para `.pdf`
- ✅ Converter `.pdf` para `.docx` (extrai texto)
- ✅ 100% gratuito, leve e sem anúncios invasivos
- ✅ Interface simples e responsiva (inspirada no iLovePDF)
